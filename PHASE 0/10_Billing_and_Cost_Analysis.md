# Billing and Cost Analysis

This document details the Azure Cloud consumption incurred by the Scientometric Analysis Tool during the processing window between March 5 and March 6, 2026.

The main cost associated with this tool comes from the use of Artificial Intelligence models (specifically Azure OpenAI models `gpt-4o` and `gpt-4o-mini`), which are used to read author affiliations and extract standardized country codes.

---

## Total Costs (March 5 - March 6, 2026)

During this period, the tool processed and extracted geolocation data from thousands of authors across multiple databases. The total billed cost for all operations during these two days was **$0.126 USD**.

The following table explains the exact consumption (measured in thousands of tokens, or 1K) and the cost in USD. "Input" refers to the text sent to the AI to read, and "Output" refers to the response generated by the AI (the country codes).

---

## Cost per Iteration

Based on the usage data from March 5th, we can calculate the exact cost of processing a single author's affiliation (one iteration).

1. On March 5th, the system processed **358,230 input tokens** using the `gpt-4o-mini` model.
2. An average affiliation string provided by the databases contains around 35 to 40 tokens.
3. This indicates that those 358,230 tokens represent the processing of approximately **8,000 to 10,000 individual authors**.
4. The total cost to process those 10,000 authors with `gpt-4o-mini` was **$0.053 USD**.

Dividing the total cost by the number of authors processed gives an estimated cost of **$0.000005 USD per iteration**. This is the cost incurred by the tool to read the affiliation string, extract the information, and output the standard geolocation for one single author.
