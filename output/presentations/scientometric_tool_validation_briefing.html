Central claim

What the validation must demonstrate

Given the same corpus-construction rules, the tool should reproduce comparable document counts, affiliation patterns, authorship-position signals, country distributions, and bibliometric indicators.

  • Differences are acceptable only when traced to database coverage, API access, deduplication, or metadata availability.
  • The validation output must include a mismatch log, not only summary figures.

Prior corpus role

How to use the prior closed DOI corpus without overstating it

  • Use it for runtime, scale, reconciliation, and figure-design stress testing.
  • Do not use it as the main scientific validation benchmark.
  • Describe it as an engineering case study only after the benchmark validation logic is established.
